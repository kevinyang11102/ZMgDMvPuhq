# 项目概述 python367

本项目是一个基于 Python + Flask 的机器学习链家网新房数据可视化及预测系统。系统通过爬虫技术获取链家全国新房数据，运用机器学习算法进行数据分析，并通过可视化工具展示分析结果。

## 技术栈

- **爬虫：** 使用 `requests` 进行数据爬取
- **数据库：** 采用 `MySQL` 存储数据
- **后端框架：** 使用 `Flask` 构建后端服务
- **机器学习预测：** 运用基于 `scikit-learn` 的多元线性回归算法进行数据预测
- **数据可视化：** 使用 `Echarts` 展示数据可视化结果

## 功能模块

- 数据爬取：自动获取链家网全国新房数据
- 数据存储：将获取的数据存入 MySQL 数据库
- 数据分析：运用机器学习算法对数据进行处理分析
- 可视化展示：使用 Echarts 展示数据的可视化图表

## 系统角色

- 用户：访问系统，查看新房数据及预测结果

## 运行环境

- Python 3.x
- Flask
- MySQL
- scikit-learn
- Echarts

## 部署步骤

1. 安装 Python 环境及相关依赖库
2. 配置 MySQL 数据库
3. 运行数据爬取脚本
4. 运行 Flask 应用
